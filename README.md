# Mapping is Boring 🗺️

Bienvenue sur le projet **Mapping is Boring** ! Ce projet contient une application interactive pour remplir des grilles à partir de données Excel.

## 📍 Accès au projet

**Site web public :** https://TheNightShiftHare.github.io/Mapping_is_boring/

## 🚀 Guide de déploiement

### Publié sur GitHub Pages
Le site est automatiquement déployé via GitHub Pages à chaque commit sur la branche `main`.

**URL du site :** https://TheNightShiftHare.github.io/Mapping_is_boring/

**Mise à jour du site :**
1. Modifiez le fichier `index.html` (ou autre fichier)
2. Commitez vos changements
3. Le site se met à jour automatiquement en quelques secondes ✅

---

## 📝 Comment mettre à jour le projet

### Pour une petite modification :
1. Allez sur le repository GitHub
2. Cliquez sur le fichier à modifier
3. Cliquez sur le **crayon** (Edit this file)
4. Modifiez le code
5. Committez en bas de la page

### Pour une modification importante :
1. Clonez le repository sur votre ordinateur
2. Modifiez les fichiers dans votre éditeur préféré
3. Testez localement en ouvrant les fichiers `.html` dans votre navigateur
4. Quand c'est bon, uploadez sur GitHub et committez

---

## 🔄 Gestion des versions

Les versions du projet sont gérées via **Git Tags**.

### Versions actuelles :
- **v7.8** : Version archivée (voir les releases)
- **v7.9+** : À venir

### Créer une nouvelle version :
1. Quand vous terminez une version stable, allez sur **Releases**
2. Cliquez **"Create a new release"**
3. Entrez le tag (ex: `v7.9`)
4. Ajoutez une description des changements
5. Publiez !

**Voir toutes les versions :** https://github.com/TheNightShiftHare/Mapping_is_boring/releases
